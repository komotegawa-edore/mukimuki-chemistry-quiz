```
認知 → 興味 → 相談 → 商談 → 契約 → 導入 → 運用 → 継続/紹介
```

## 3. 相談フェーズ

### タッチポイント
- 電話ヒアリング（15-30分）
- Zoom相談（希望者）

### ヒアリング項目
- [ ] 塾の規模（生徒数、講師数）
- [ ] 現在のHP有無・状態
- [ ] 集客の課題
- [ ] 希望するデザインの方向性
- [ ] 予算感
- [ ] 導入希望時期

### ゴール
- 訪問アポイントの獲得
- または電話/Zoomでの契約

---

## 4. 商談フェーズ

### タッチポイント
- 訪問商談（60分）
- Zoom商談（30-45分）

### 商談の流れ

```
1. アイスブレイク（5分）
   - 塾の雰囲気を褒める
   - 共通の話題を見つける

2. ヒアリング深掘り（15分）
   - 現在の集客方法
   - HPへの不満・要望
   - 競合塾の状況

3. 現状診断（10分）
   - 現在のサイトをスマホで確認
   - 「地域名 塾」で検索結果を見せる
   - 問題点を指摘

4. 提案（15分）
   - JUKUBAのデモ
   - テンプレート選択
   - 管理画面の操作デモ
   - 「自分で更新できる」を実演

5. 料金説明（10分）
   - 月額2,980円〜を強調
   - 初期費用は規模・要望による
   - 競合（制作会社30万〜）との比較

6. クロージング（5分）
   - 質疑応答
   - 契約の意思確認
   - 次のステップ説明
```

### 料金の伝え方

```
「月額は2,980円からです。
 初期費用はお客様の規模やご要望によって変わりますが、
 通常の制作会社さんだと30万〜50万かかるところを、
 JUKUBAなら3万円〜10万円程度でスタートできます。

 まずはご要望をお聞きして、お見積りをお出ししますね。」
```

### ゴール
- 契約の意思確認
- 見積書の提出約束

## 6. 導入フェーズ（※ここでアカウント作成）

### 運営側でやること

```
入金確認後:
1. アカウント作成（運営側で実施）
2. 初期設定（テンプレート適用）
3. 基本情報入力代行
4. サイト公開
5. ログイン情報を顧客に引き渡し
```

### タイムライン（2週間）

| 日 | 作業 | 担当 |
|----|------|------|
| Day 1-2 | 素材収集依頼 | 運営 |
| Day 3-5 | 素材提供（写真、文章） | 顧客 |
| Day 6-8 | サイト構築、初期設定 | 運営 |
| Day 9-10 | 内容確認、修正依頼 | 顧客 |
| Day 11-12 | 修正対応 | 運営 |
| Day 13 | 最終確認 | 顧客 |
| Day 14 | 公開 & 引き渡し | 運営 |

### 引き渡し時にやること
- [ ] ログイン情報の共有
- [ ] 管理画面の使い方レクチャー（30分）
- [ ] よく使う操作のマニュアル提供
- [ ] 困ったときの連絡先共有

### ゴール
- サイト公開
- 顧客が自分で更新できる状態

## アカウント作成フロー（運営側）

### なぜ運営側で作成するか
1. **契約確定後のみ作成** → 無料アカウントの放置防止
2. **初期設定とセット** → 引き渡し時に完成状態
3. **品質管理** → 適切な設定がされた状態で開始
4. **サポート体制** → 最初から関係性構築

### 作成手順（運営側）

```
1. 顧客情報を管理画面で入力
   - 塾名
   - オーナー名
   - メールアドレス
   - 電話番号

2. アカウント作成（Supabase Admin）
   - ユーザー作成
   - juku_owner_profiles にレコード追加

3. サイト作成
   - テンプレート選択
   - 基本情報入力
   - カラー設定

4. 仮パスワードでログイン情報を顧客に送付
   - 初回ログイン時にパスワード変更を促す
```

## LPの改善ポイント

### 現状の問題
- 初期費用が目立っている
- 「無料で始める」ボタンがあるがセルフ登録不可

### 改善案

1. **月額を最前面に**
   ```
   月額 2,980円〜
   塾専用ホームページ
   ```

2. **初期費用は「要相談」に**
   ```
   初期費用: ご要望に応じてお見積り
   ```

3. **CTAを「無料相談」に統一**
   ```
   「今すぐ無料で作成」→「まずは無料相談」
   ```

4. **セルフ登録ボタンを削除**
   - /juku-admin への直接リンクを削除
   - すべて問い合わせフォームへ誘導
