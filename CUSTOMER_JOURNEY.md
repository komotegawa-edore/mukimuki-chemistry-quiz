# Roopy カスタマージャーニーマップ（生徒向け）

| 項目 | 内容 |
|------|------|
| **名前** | 田中 陽太（仮名） |
| **年齢** | 17歳（高校2年生） |
| **志望校** | 国公立大学（中堅〜上位） |
| **現状** | 何から始めればいいかわからない。勉強時間は確保しているが成績が伸びない |

## ジャーニーフェーズ概要

```
認知 → 興味 → 検討 → 利用開始 → 習慣化 → 課金 → 推奨
```

😟 不安・焦り・迷い

### 感情

### ユーザーの疑問

- 「他のアプリと何が違う？」

### タッチポイント

| **TikTokプロフィール** | アプリへのリンク |

- [ ] ユーザーレビュー誘導
- [ ] LP作成（roopy.app）

### KPI

## Phase 3: 検討・ダウンロード（Consideration）

### ユーザーの状況
- アプリをダウンロード
- 初回起動、オンボーディング開始

### 感情
✨ 期待・ワクワク

### ユーザーの行動
1. App Storeでダウンロード
2. アプリ起動
3. オンボーディング画面を見る
4. アカウント作成（または後回し）

### オンボーディングで伝えるべきこと
1. **価値提案**: 「あなた専用の勉強計画を30秒で作成」
2. **簡単さ**: 「3つの質問に答えるだけ」
3. **即効性**: 「今日やることがすぐわかる」

### 必要なUI/UX
- [ ] シンプルなオンボーディング（3画面以内）
- [ ] スキップ可能なアカウント作成
- [ ] 初回ロードマップ生成への誘導

### KPI
- DL→初回起動率: 80%以上
- オンボーディング完了率: 70%以上

## Phase 4: 利用開始（Activation）

### ユーザーの状況
- 初めてロードマップを生成
- 「おお、これいいかも」と感じる（Aha Moment）

### 感情
🎉 感動・驚き・「これ使えそう」

### Aha Moment（成功体験）
> **「自分の志望校に合った計画が自動で作られた！」**

このモーメントを早く体験させることが最重要。

### ユーザーの行動
1. 志望校・現在の学力を入力
2. 受験日までの日数を確認
3. **ロードマップが生成される** ← Aha Moment
4. 「今日やること」を確認

### 必要な機能
- [ ] 3ステップでロードマップ生成
- [ ] ロードマップの視覚的表示（ガントチャート風）
- [ ] 「今日やること」の明確な表示
- [ ] 生成後の「シェア」ボタン（口コミ促進）

### KPI
- ロードマップ生成率: 60%以上（DLユーザー中）
- 生成後24時間以内の再訪問率: 50%以上

## Phase 5: 習慣化（Retention）

### ユーザーの状況
- 毎日アプリを開いて「今日やること」を確認
- タスクを完了としてチェック
- 進捗を実感

### 感情
😊 達成感・安心感・モチベーション維持

### 習慣化のトリガー
| トリガー | 内容 |
|----------|------|
| **時間トリガー** | 毎朝/毎晩の通知 |
| **完了トリガー** | タスク完了時のフィードバック |
| **進捗トリガー** | 週次レポート・達成率表示 |

### 必要な機能
- [ ] プッシュ通知（朝: 今日のタスク / 夜: 振り返り）
- [ ] タスク完了時のポジティブフィードバック
- [ ] 連続利用日数（ストリーク）表示
- [ ] 週次進捗レポート
- [ ] 達成バッジ・実績機能

### 離脱防止ポイント
| 離脱リスク | 対策 |
|------------|------|
| 計画通りいかない | 計画の再調整機能 |
| モチベ低下 | 応援通知、名言表示 |
| 使い方がわからない | ヘルプ、チュートリアル |
| 飽きる | ゲーミフィケーション |

### KPI
- Day 1 リテンション: 40%以上
- Day 7 リテンション: 25%以上
- Day 30 リテンション: 15%以上
- 週次アクティブ率: 30%以上

## Phase 6: 課金（Conversion）

### ユーザーの状況
- 無料版の制限に到達
- 「もっと使いたい」と感じている
- 有料版の価値を検討

### 感情
🤔 「お金払う価値ある？」→ 💳 「これは投資だ」

### 無料→有料の壁を越えるトリガー
| トリガー | 詳細 |
|----------|------|
| **制限到達** | ロードマップ保存不可、再生成不可 |
| **価値実感** | 1週間使って「これ便利」と感じた |
| **タイミング** | 模試前、新学期、夏休み開始 |
| **価格比較** | 「塾より安い」「参考書1冊分」 |

### 課金ポイントの設計

```
無料版の制限:
- ロードマップ生成: 1回のみ（保存不可）
- ガントチャート: 閲覧のみ
- PDF出力: 不可
- 過去の進捗: 7日分のみ

有料版で解放:
- ロードマップ無制限生成・保存
- ガントチャート編集
- PDF出力
- 全履歴閲覧
- プッシュ通知カスタマイズ
```

### 課金導線
1. **自然な導線**: 制限到達時に「もっと使うには」表示
2. **価値訴求**: 「月1,000円で合格への最短ルート」
3. **比較**: 「塾の1/10以下の価格」
4. **無料トライアル**: 7日間有料機能お試し

### 必要なUI
- [ ] 課金ページ（シンプル、2プラン表示）
- [ ] 機能比較表
- [ ] 「お得」バッジ（年額プラン）
- [ ] 決済失敗時のリカバリーUI

### KPI
- 無料→有料転換率: 10%
- トライアル→有料転換率: 50%
- 課金後1ヶ月継続率: 80%
